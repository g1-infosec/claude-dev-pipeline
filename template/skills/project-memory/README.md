# project-memory

The team's living memory, and the thing that keeps you from getting lost between sessions.

- **Always on.** Every agent reads it before working and updates it after.
- **Three files** under `.claude/memory/`: `PLAN.md` (what's next), `DECISIONS.md` (why you chose things), `PROGRESS.md` (plain-language status).
- Come back after a week and `PROGRESS.md` tells you exactly where you left off.

Details in [`SKILL.md`](./SKILL.md).
