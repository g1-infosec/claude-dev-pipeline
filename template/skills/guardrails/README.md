# guardrails

The seatbelt against self-inflicted damage. Always on, in every mode.

- Stops secrets being committed; ensures a `.gitignore` and `.env.example` exist.
- Nudges you onto a branch instead of `main`, and to commit before risky work.
- Requires a backup (or an explicit "yes, I know") before destructive migrations or deletions.

Details in [`SKILL.md`](./SKILL.md).
